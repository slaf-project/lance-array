# lance-array

Chunk-aligned helpers for **2D Zarr 3** arrays and **Lance blob v2** tiles: `ZarrChunkTileArray`, `LanceBlobTileArray`, and `write_lance_blob_tile_dataset`.

See the repository [README](https://github.com/slaf-project/lance-array) for install and benchmark instructions.
